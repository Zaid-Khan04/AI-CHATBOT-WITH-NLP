# AI Chatbot with NLP

## Description
An intelligent chatbot built using NLTK and SpaCy that understands natural language, recognizes entities (names, locations), and maintains conversation history with timestamps.

## Features
- Natural language processing using NLTK and SpaCy
- Entity recognition (names, locations, organizations)
- Pattern-based responses with smart context awareness
- Conversation logging with persistent history
- Sentiment detection

## Installation

1. Install required libraries:
```bash
pip install nltk spacy
python -m spacy download en_core_web_sm
```

2. Run the chatbot:
```bash
python chatbot.py
```

## Usage
```
You: Hello
Bot: Hi there! What can I do for you?

You: My name is Zaid
Bot: Nice to meet you, Zaid! What can I do for you?

You: I am from Mumbai
Bot: Oh, Mumbai! That's a wonderful place! How can I help you today?
```

## Technologies
- Python 3.x
- NLTK (Natural Language Toolkit)
- SpaCy (Advanced NLP)

## Output
- `conversation_log.json` - Stores all conversation history
